📄 Resume Screening & Candidate Ranking System

🚀 Machine Learning Task 3 (2026) – Future Interns

🔍 About the Project

Hiring teams often receive hundreds of resumes for a single job role, making manual screening:

⏳ Time-consuming

❌ Inconsistent

⚠️ Error-prone

👉 This project builds an AI-powered Resume Screening System that automatically:

Extracts skills from resumes

Matches them with job requirements

Ranks candidates based on role fit

Identifies missing skills

This type of system is widely used in HR-tech platforms, recruitment tools, and hiring systems.

🎯 Objective

The goal is to build a system that can:

📄 Read resume text (PDF / text format)

🧠 Extract skills and keywords using NLP

📋 Compare resumes with job descriptions

📊 Rank candidates based on relevance

⚠️ Highlight missing or required skills

💡 Outcome:

Helps recruiters shortlist candidates faster and more accurately.

✅ Features Implemented

✔ Resume Text Cleaning & Preprocessing
✔ NLP-based Skill Extraction
✔ Job Description Parsing
✔ Resume-to-Job Similarity Matching
✔ Candidate Ranking System
✔ Skill Gap Identification

⭐ Bonus Features:

Weighted skill importance

Candidate comparison visualization
